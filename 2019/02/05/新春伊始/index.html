新年伊始

发表于 2019-02-05 | 分类于 杂谈 |

爆竹声中一岁除，春风送暖入屠苏。

——2019.02.05 正月初一晚


        时间过得真快啊~不过每次想写点啥的时候都想说这句话，O(∩_∩)O哈哈~本来这个月应该在上个月底写总结的。但是想着过年也在二月初，于是就想着一块写了~

        翻了翻上个月的计划，除了李宏毅老师GAN的视频没看之外，其他的多多少少都完成了些(但是好像其他的也没啥(╯▽╰ ))。总的来说，上个月主要还是在调CVPR rebuttal的实验，从SNGAN STL10的实验，换到了现在GAN_stability CelebA的实验，但是最后的效果还是不好，其实认真反思，可以用潘老师的一句话总结：

代码都给你了，github上也有参考代码，这样不难了吧？

        说实话，虽然这几年来，自己的自尊心已经打磨的所剩无几了。但是这句话还是让自己感觉到很羞愧。但是确确实实，自己的心还不够沉静，还不能全身心投入一件事件中。这本是一件不费多少工夫的事情，可到最后半个月了，还是卡在这里不能前进。希望自己在这其中能够有所反思，有所得。经过这半个月的试探，原因也快要浮出水面了，坚持住。

        闲话少说，还是按照惯例留下这个月的计划，这个月时间比较短，而且二月二十五日才正式上课，因此这个月的计划相应地留的简短一些。

  • 第一是继续找GAN_stability CelebA的实验崩溃的原因，但要注意平衡好时间，并且全身心投入。这次实验结束之后，总结一下代码，然后写一篇博客总结一下。
  • 第二是蚂蚁金服那个项目开始着手重构了，暂时按照现在的分工，我主看论文，之后代码再两个人一块分析。
  • 第三是继续看paper，具体是什么样的paper，暂时不限制，但是要注意看paper的方法，不要拖拉，做好笔记。
  • 第四是这个月看两场电影，然后计划好下学期的选课。
谢谢你请我吃糖果(*^▽^*)
